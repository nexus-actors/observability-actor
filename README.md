# nexus-actors/observability-actor

Nexus actor-system observability — internal-state metrics (live actors, dead letters, running) as OTEL gauges.

> **Read-only subtree split** of [nexus-actors/nexus](https://github.com/nexus-actors/nexus).
> Report issues and send pull requests to the monorepo — this repository only receives
> automated pushes and release tags.

## Installation

```bash
composer require nexus-actors/observability-actor
```
